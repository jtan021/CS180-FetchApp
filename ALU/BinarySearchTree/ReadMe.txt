Binary Search Tree

Leader: 
Jonathan Tan

Members:
None

Remarks:
An implementation of a Binary Search Tree with the following functions:
size() 		// Returns number of nodes
empty() 	// Returns 1 if Size() == 0, else 0
print_node()	// Prints value stored within node passed into parameter
height()	// Returns height of tree
search()	// Searches the bst for a value passed in as a parameter and returns 1 if found, 0 otherwise
PRE_Output()	// Helper function for preorder()
preorder()	// Outputs values of the tree in PREOrder format
POST_Output() 	// Helper function for postorder()
postorder()	// Outputs values of the tree in POSTOrder format
IO_Output()	// Helper function for inorder()
inorder()	// Outputs values of the tree in INOrder format
set_inorder() 	// Pushes values of the tree onto a list in INOrder format
operator[]	// Overrides [] operator for bst
insert()	// Inserts value passed in as parameter into the tree
remove()	// Removes node with value passed in as paramater from the tree
minCoverRecursion() // Pushes node values onto a list passed in as the second parameter from smallest to largest value
minCover()	// Helper function for minCoverRecurssion()
displayMinCover() // Outputs the list made by minCoverRecursion
...		// More functions added to be described in ReadMe.
