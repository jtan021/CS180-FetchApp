CPlusPlus Shell Builder
=======================
Basic C++ script builder which creates file  <Class>.cc and header file <Class>.hh based on input parameters. 

Main Script: ./InitClassFiles <Class Name> <Member1:Data Type> <Member2:DataType> ... <MemberN:DataType>
Description: Main shell file which utilizes the other shell files in the directory to construct a basic C++ script.
